Media Browser

The Media Browser is a simple utility designed to quickly browse your photo libraries. It also brings a familiar and powerful browser experience for links.

General Tips

    Once launched, the Media Browser can be accessed via its icon in the dock; it will hide when it is no longer the active application.

    To access application menus (Preferences) when using the  status bar icon, right click on the icon.

    Click the  button in the upper right corner of the Media window, or choose "About Media Browser…" from the application menu, to change the window to reveal preferences for toolbar appearance and display of captions.

    Any images types allow you to drag in your own folder of files into the top panel.

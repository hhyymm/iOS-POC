Media Browser Help

The Media Browser is a simple utility designed to quickly browse your photo libraries. It also brings a familiar and powerful browser experience for links.

General Tips

Once launched, the Media Browser can be accessed via its icon in the dock; it will hide when it is no longer the active application. Using the Preferences window, you can keep the application visible at all times, or access Media via the icon in the status bar (to the right of the menu bar).

To access application menus (Preferences) when using the status bar icon, right click on the icon.

Use the Preferences window to set general application behavior, including window and dock appearance.

Click the button in the upper right corner of the Media window, or choose "About Media Browserâ€¦" from the application menu, to change the window to reveal preferences for toolbar appearance and display of captions.

Many media types allow you to drag in your own folder of files into the top panel.


Modules

Photos

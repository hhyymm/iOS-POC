Preferences

Displaying Media Browser

When you initially launch Media Browser, it's available via the Dock. You can also change the application to be available via the "Status Bar" (to the right of the menu bar). This change will not take effect until the next time you launch the application. (Technical note: Because this choice changes a setting within the application package itself, you will need write permissions on the application to successfully make the change.)

Window Visibility

You can set the Media Browser to hide itself when it is no longer the active (frontmost) application, or to be visible at all times.
